发表于 2023-05-28 更新于 2023-07-18 分类于 随笔

不可放任大脑胡思乱想，但大脑又岂非人能够驾驭得了的。放任大脑，便会增加精神内耗，精力内耗。孩子为何时时精力充沛，也和此种相关，成人胡思乱想范围广，时间长，大脑的运行和现实就恰似天上一天，地上一年，有过之而无不及。
由此可知，减少大脑胡思乱想是减少精神内耗，保持精力的有效手段！
既然大脑幻想不能人为驾驭，但又不能放任其消耗精力，那就思考如何人为干预。
在认真学习工作期间（认真，注意力集中），完全投入进去。以及打游戏期间等等诸如此类大脑的幻想是停止的，至少未意识到有什么幻想在其中。由此看来，消耗替代的方式是无效的，因为虽然不会幻想，但同样消耗精力，这里要思考的是弱化，减少，不可能完全禁止，哪怕睡觉大脑幻想这一部门也是在运行中的，虽然睡觉能够恢复精力，但是人又不能时刻睡觉。无论正向反向的幻想都会让人产生自卑苦闷等等情绪，只不过反向是进行中的情绪，正向是幻想解除后现实无奈产生的情绪。
那么是否可以思考一种引导的幻想，减少精神内耗，减少精力消耗。
发呆可以，发呆就像大脑宕机，但不可能时刻发呆，也不能人为控制。做一些让人放松的事，快乐的事。心情不愉悦，精力消耗的就越快，身体器官就越疲倦，就越不健康，就越短命。
生物本能，各个器官本能都是生存下去，大脑胡思乱想消耗精力却会起反作用，是否意味着幻想产生的细胞也想生存下去，而每次幻想结束，即意味着某些细胞将要死亡。想的多，死的快。
就像社会里的人，地球里的生物，为了生存就要去伤害另一部分，斗争过程反而加速消耗资源。最终短命。似乎这就是真理，这是真理吗？不愿意去相信。
少幻想，如何少？强迫自己背一些古诗词。
